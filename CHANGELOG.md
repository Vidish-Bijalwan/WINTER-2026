# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [Unreleased]

### Added
- Initial project structure with React frontend and Python backend
- Real-time Wikipedia edit stream ingestion via SSE
- Topological Data Analysis (TDA) engine for anomaly detection
- 3D Network visualization using Three.js
- Dashboard with Recharts for statistical monitoring
- Docker support for containerized deployment
- GitHub Actions CI/CD pipeline
- Comprehensive documentation (API, Deployment, Contributing guidelines)

### Changed
- Migrated from generic template to custom TopoShape Insights branding
- Optimized TDA algorithms for client-side performance

### Fixed
- Resolved memory leaks in long-running 3D visualizations
- Fixed WebSocket reconnection logic for unstable connections
